# Security

Supericons MCP is designed as a read-only icon search service.

The free local setup starts the public `@supericons/mcp` npm package through `npx`. It does not need a Supericons account or API key.

For Pro tools or premium icon access, use `SUPERICONS_API_KEY` as an environment variable. Keep API keys out of screenshots, public issues, shared config files, and source control.

Do not submit private keys, service role keys, database credentials, or customer data in bug reports.

To report a security concern, contact Curly Mole Labs through the official Supericons website:

https://supericons.dev
